A Few Projects I'm Proud Of

Perlin Noise Flow Field

With this project, I used JavaScript to create a flowfield that directs particles according to a series of vectors. The vectors change angles according to Perlin Noise, creating a very natural flow from angle to angle. Because Perlin Noise is used, the flow field will create a new piece of artwork each time the page is loaded. If you refresh the page, you'll notice different shapes being drawn each time.

Machine Learning Body Recognition

Click/Tap the screen to turn the ligths on and off. This project uses machine learning to detect and plot joints in order to represent a body on screen. This can be used as an input for several applications, like a game where you jump to make the character jump and crouch down to make the character crouch down.
